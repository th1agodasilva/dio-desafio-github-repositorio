 # Principais Comandos Bash

## Comandos Básicos

* cd

* dir

* mkdir

* del/rmdir

* ctrl + l

  | Comandos  | Funções                                                      |
  | :-------- | ------------------------------------------------------------ |
  | cd        | Mudar de diretório                                           |
  | dir       | Exibe uma lista de arquivos e subdiretórios em um diretório. |
  | mkdir     | Criar diretórios                                             |
  | del/rmdir | Remover um diretório.                                        |
  | ctrl + l  | Limpar a tela do terminal.                                   |
